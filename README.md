## Dependencias
* Python
* Tk
* Graphviz
* Pillow

## Instalar dependencia
Por PowerShell(Windows) o por Terminal(Linux/Mac)
```pip install tk```

```pip install graphviz```

```pip install pillow```

## Run
```python AutoEd.py```

## Compilar con pyinstaller
Instalar pyinstaller por
```pip install pyinstaller```

Y compilar por
```pyinstaller .\AutoEd.py```
